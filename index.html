GsDevKit Projects

These are available projects that you may install and load into your GsDevKit environment.
Installing registers components on disk in your GsDevKit environment; loading involves loading the code into a Stone.
You must have a GsDevKit installation with the tODE client running in order to perform the tODE installation.
Project tODE load instructions

Seaside

View Seasidest/Seaside project 
project install --url=http://gsdevkit.github.io/GsDevKit_home/Seaside32.ston
project load --loads=`#('Welcome' 'Development' 'Zinc Project' 'Examples')` Seaside3
